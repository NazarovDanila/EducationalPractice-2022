Скрипт ms воссоздаёт ТОЛЬКО таблицы базы данных LopushokDB. 
Сама база данных уже должна быть создана.
Данные в базу нужно вносить самостоятельно. 
В качестве источника можно использовать Excel файлы.
В названии файла указана таблица, в которую нужно внести данные из файла.
Рекомендуется заполнять таблицы в порядке нумерации файлов.


Скрипт LopushockScript воссоздаёт базу данных целиком, со значениями в таблицах, 
которые можно заполнить, используя изначальные материалы задания.
Используется для проверки.
